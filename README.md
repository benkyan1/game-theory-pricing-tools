# Pricing Strategy Analytics

A collection of Jupyter notebooks for pricing analysis, game theory simulation, and competitive strategy.

## What's Inside

| Notebook | What it does |
|----------|--------------|
| `game_theory_pricing_simulator.ipynb` | Interactive tool for Bertrand competition and Nash equilibrium |
| `price_sensitivity_models.ipynb` | Methods to measure price elasticity (regression, IV, Double ML) |
| `switching_cost_models.ipynb` | Ways to quantify customer switching costs from data |
| `brand_strength_models.ipynb` | Approaches to measure brand value and loyalty |

Use Cases
Find optimal pricing under competition

Calculate true price elasticity

Estimate how much customers pay to switch

Quantify brand value in dollars
